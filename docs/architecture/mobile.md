# Mobile

## Tech Stack

- **Flutter** >=3.3.0
- **Riverpod 2.5** — State management
- **GoRouter 14.2** — Navigation
- **Dio 5.6** — HTTP client
- **Flutter Secure Storage** — Token storage
- **Firebase Cloud Messaging** — Push notifications
- **Sentry Flutter** — Error tracking
- **Connectivity Plus** — Offline detection

## Features

- Login via Cognito PKCE
- Dashboard com métricas
- CRUD de casos
- Gestão de prazos
- Upload e visualização de documentos
- Notificações push (Firebase)
- Modo offline (SharedPreferences)
- Perfil e configurações
